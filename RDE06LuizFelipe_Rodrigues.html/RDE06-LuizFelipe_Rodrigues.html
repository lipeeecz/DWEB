SOFTWARES DE VERSIONAMENTO

Versionamento de software é o processo de atribuir um nome único ou uma númeração única para indicar o estado de um programa de computador.
Esses números são geralmente atribuídos de forma crescente e indicam o desenvolvimento de melhorias ou correção de falhas no software.

GITHUB

É uma plataforma de hospedagem de um código-fonte e arquivos com controle de versão.

GIT

Repositório com histórico completo e não dependente de acesso a uma rede ou a um servidor central para revisões e acompanhamento

GITHUBDESKTOP

É um aplicativo que permite que você interaja com o GitHub usando uma GUI em vez da linha de comando ou de um navegador web
